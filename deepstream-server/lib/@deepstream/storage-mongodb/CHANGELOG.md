# Compatibility issues / Breaking Changes
- deepstream metadata is now stored within each record under `__ds` to improve querying. Updating to this version will require data to be migrated to new structure.

Now
```json
{
   "name": "John'",
   "age": "24",
   "__ds": {
     "_v": "10"
   }
}
```

Before:
```json
{
     "_v": "10",
   "_d": {
      "name": "John",
      "age": "24"
   }
}
```
